# Grazi Imports

E-commerce demo for beauty products imported from the USA.

## Run

Open `index.html` in a browser (double-click or live server). No build step required.

## What’s included

- **Shop** — catalog, search, filters, sort, product detail, wishlist, cart
- **Admin** — add / edit / remove products, live stock control, orders, coupons
- **Checkout** — PIX (5% off), card, boleto, WhatsApp payment (in-memory demo)
- **WhatsApp** — floating button, cart order message, product inquiry
- **Social** — Instagram grid, TikTok / Facebook / Pinterest links, newsletter

All state is **in-memory MVC** (Model / View / Controller in one file) for UI validation — refresh resets data.

Replace `WA_NUMBER` in the script with your real WhatsApp number before production use.
